# Todo Application
Technical stack:
- TypeScript
- React
- Redux
- redux-saga
- redux/toolkit

## Live Demo
[https://todo.galishnikov.ru/](https://todo.galishnikov.ru/)


## Installation
```sh
git clone git@github.com:galishmann/todo.git
cd todo
npm install
npm start
```

Application will be started on http://localhost:8080
